#React Quiz APP

## A simple quiz app build with ReactJS

###Info:

- a simple ReactJS Quiz App with results.

###Frameworks & Languages used:

- ReactJS
- NodeJS
- JavaScript (ES6, ES2015)
- HTML5/CSS3
- JSON
- jQuery
- AJAX
- Bootstrap
- Webpack

###Demo:

http://react-quiz-app.bitballoon.com/


###Installation

```
$ npm install

$ npm start

$ open http://localhost:3001

```

###Deployment

```
$ bash deploy

```
